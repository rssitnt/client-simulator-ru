# AI Agent Studio

Студия для создания и тестирования AI-агентов на базе Google Gemini API.

## Возможности

- **Разделенный интерфейс**: Чат для тестирования слева, редактор промпта справа
- **Системный промпт**: Настройка поведения AI-агента
- **Настройки генерации**: Температура и максимальное количество токенов
- **Сохранение настроек**: Автоматическое сохранение промпта и настроек в localStorage
- **История диалога**: Контекстные диалоги с сохранением истории
- **Минималистичный дизайн**: Темно-серый интерфейс

## Запуск

Просто откройте `index.html` в браузере. Никаких дополнительных зависимостей не требуется.

## Использование

1. **Настройте системный промпт** в правой панели - это инструкции для вашего AI-агента
2. **Настройте параметры генерации**:
   - Температура (0-2): Контролирует креативность ответов
   - Макс. токенов: Максимальная длина ответа
3. **Начните диалог** в левой панели - вводите сообщения и тестируйте поведение агента
4. **Нажмите "Сохранить"** для сохранения настроек промпта
5. **Используйте "Очистить"** для начала нового диалога
