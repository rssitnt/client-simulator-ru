## Суть
Сайт, где ИИ играет роль покупателя навесного оборудования, а менеджер отрабатывает кейс.
После этого другой ИИ оценивает менеджера по внутренней структуре оценки.

## Главные задачи
- надёжность
- безопасность
- логика сценариев
- качество оценки
- общий UX
